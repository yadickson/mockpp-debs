# mockpp-debs
Build Debian Package for mockpp

**Build dependencies**

- debhelper (>= 9)
- dh-autoreconf (>= 2)
- autotools-dev
- libcppunit-dev (>= 1.12.1)

**Tested**

- Debian wheezy
- Debian jessie

- [Home page from source code](https://sourceforge.net/projects/mockpp)
